# Lab 1 - Joseph Haggerty

## Introduction

This code will display a basic webpage about the Fibonacci Numbers as per the requirements of Lab 1. A JavaScript script handles the calculation of the Fibonacci Numbers.

## Files

The necessary files to run this webpage are as follows:
* fib.js
* fib.png
* jmh463_lab1.html

## Installation

Simply place the previously mentioned files into a folder of your choosing and open the .html file with any web browser. All three must be in the same folder.

## Using the Webpage

After opening the webpage in a browser, you can read basic information on Fibonacci numbers and get a Fibonacci value using the form at the bottom of the page. Just type in a non-negative integer and press "Calculate." The nth value and a table of the sequence will be displayed below. Invalid input will show a warning below instead.
