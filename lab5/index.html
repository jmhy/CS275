Welcome to Lab 5 of CS 275!

The purpose of this lab is three-fold:

  1. Getting experience with MySQL databases, including how to install/set up a database and creating/populating relational tables through the MySQL command line interface
  2. Interfacing with a MySQL database through a Node.js server
  3. Gaining more experience in designing an application/website

This lab has the following dependencies:

  • jQuery (3.1.1 or later)
  • MySQL Community Server
  • Node.js
  • npm modules:
    • body-parser
    • express
    • mysql

Using This Page

First, import the schema from the DumpLab5Schema.sql file into your database with the command:

"mysql -u <username> -p < DumpLab5Schema.sql"

The main page, which is the file "index.html," should not be opened directly in a browser. Instead, open a command-line, such as Command Prompt, navigate to the directory containing server.js, and run the command "node server.js" without quotes. Next, open a browser and type the URL "localhost:8080" as the server is set to listen to port 8080. The server will then display this page and you can begin using the links in the navigation area to the left. Click "HOME" to return here anytime.

Notes

This page is set up so that the banner and navigation areas are fixed to the top and left, respectively. Since there is not much content to consume space, it might not be noticeable on larger monitors.
