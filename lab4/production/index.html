Welcome to Lab 4 of CS 275!

The purpose of this lab is three-fold:

  1. Allowing for multiple pages with a mixture of static and dynamically generated content.
  2. Beginning to apply good software design principles by segmenting logic into their own modules.
  3. Getting experience from playing with Node.js emitters.

This lab integrates content from both labs 2 and 3, which involved the following:

  • Lab 2 - Learn how to utilize jQuery and Ajax to develop a web page to invoke the Weather Wunderground web service and parse its JSON response
  • Lab 3 - Two-fold:
    • Set up and test a server side environment
    • Get acquainted with Node.js basics, including interactions between client and server along with the development of a simple dynamic web service

Using This Page

The main page, which is the file "index.html," should not be opened directly in a browser. Instead, open a command-line, such as Command Prompt, navigate to the directory containing server.js, and run the command "node server.js" without quotes. Next, open a browser and type the URL "localhost:8080" as the server is set to listen to port 8080. The server will then display this page and you can begin using the links in the navigation area to the left to try out the calculation and weather services. Click "HOME" to return here anytime.

Notes

This page is set up so that the banner and navigation areas are fixed to the top and left, respectively. It might not be noticeable on larger monitors.
